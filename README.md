**Обнаружение (детекция) дронов на поступающем фото/видео потоке**

1. Проект представляет собой модуль компьютерного зрения для детекции дронов в видеопотоке или на статичных изображениях с использованием модели YOLO8n и библиотек OpenCV, torch, torchvision, написанный на python v3.12.3. Система работает в режиме реального времени и с задействованием CUDA при его наличии на устройстве.
2. Функционал проекта:
   - Загрузка предобученной модели YOLO (весов и конфигурации) "best.pt". За основу взята предобученная модель YOLO8n, над которой проведено дообучение открытым датасетом Deadly-flying-bois-yt-2 циклом в 100 эпох.
   - На вход модели для прогнозирования попадает один из видов контента: изображение, видео из локальной директории, видео из веб-камеры. При этом размеры подгоняются под нужные для использования моделью (с целью сохранения пропорций, длинная сторона доводится до размера 640, короткая дополняется до 640 нулями) для устоячивости пропорций и сохранения всего изображения.
   - Происходит детекция объектов класса "drone" с фиксацией координат и фильтрации его по порогу вероятности уверенности детектирования. В процесс по умолчанию для YOLO8n уже включена процедура non-maximum suppression (NMS).
   - Осуществляется обработка кадра с добавлением рамки вокруг объекта, названием класса детектирования и значения уверенности модели в прогнозировании детекции. Также для видео добавляется информация по количеству кадров в секунду.
   - Результирующее изображение (последовательность изображений для видео) отображаются в режиме реального времени. Поток изображений от видео можноостановить в любой момент, нажав клавищу "q" на клавиатуре.
   - В консоль по завершению обработки выводится информация об обнаруженных моделью детекциях в JSON формате.
3. Модель можно дообучать, используя уже имеющиеся гиперпараметры из файла весов и конфигурации.
4. Для запуска программы необходимо в консоли терминала выполнить команду в трех вариациях:
   - python main.py --source webcam
   - python main.py --source video.mp4
   - python main.py --source image.jpg
